# My css framework

## How to use ?

* Link "css_framework.css" in the head of your index:

``` html
<link rel="stylesheet" href="./framework_css/css_framework.css"/>
```

## Color options :

1. Backgrond color:

```html
<div class="
.bg-muted
.bg-primary
.bg-succes
.bg-info
.bg-warning
.bg-error"
>
```

2. Text color:

```html
<div class="
.text-muted
.text-primary
.text-succes
.text-info
.text-warning
.text-error"
>
```

## Button options :

1. Color

```html
<button class="
.btn-blue

.btn-green
.btn-purple
```

2. Size

```html
<button class="

>
```
